# Network Anomaly Detection

## Overview
This project builds an unsupervised machine learning system to detect anomalous
network traffic that may indicate malicious activity.

## Objectives
- Analyze network traffic data
- Identify abnormal behavior using anomaly detection algorithms
- Compare different models
- Visualize anomalies in network data

## Planned Methods
- Isolation Forest
- DBSCAN
- One-Class SVM

## Dataset
NSL-KDD dataset will be used for detecting network intrusions.
